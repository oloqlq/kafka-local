               |          |
               |          | [ CHANNEL A: DIRECT BYPASS ]

               |          |                                                         |

TRANSFORM    : (A: Bypass)  +--[ Grok    : Pattern Matching & Extract Fields   ]--+ |
               |            +--[ Mutate  : Data Type Conversion (Str->Float)   ]--+ |
               |            +--[ Tagging : Add Metadata for Conditional Alerts ]--+ |
               |                                                              |     |

               |          |             ( Merge Structured & Processed Data )
